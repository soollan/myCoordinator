4. 운영자는 새로운 브랜드를 등록하고, 모든 브랜드의 상품을 추가, 변경, 삭제할 수 있어야 합니다. 이 4가지 기능을 사용할 수 있게 하는 4개의 API를 구현해야 합니다.
